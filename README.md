# Raspi-video-to-PC
Transmit the real time picture to PC by raspi camera by socket

#Instructions
1. Enable your picamera
2. check your pc ip 
3. use command v4l2-ctl --list-devices check your picamera device number
4. Run serve.py first and then run client.py
